O que é o Check-in Colorado?

Check-in Colorado é um app para Android e iOS para a coloradagem fazer check-in para partidas no Beira-Rio pelo celular! É uma iniciativa nascida no Grupo Gigante Para Sempre, e mantida gratuitamente pelos seus colaboradores!
